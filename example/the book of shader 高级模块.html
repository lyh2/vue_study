mat3 brightnessContrast(float b, float c) {
  return mat3(c,0,0, 0,c,0, 0,0,c) + b;
}

1.3 Kernel Convolutions（卷积核）
1.0
🧮 3×3 卷积核矩阵
⎡ a   b   c ⎤
⎢ d   e   f ⎥
⎣ g   h   i ⎦
输出像素 = sum(核 * 邻域像素) / 归一化
每个输出像素是其 3×3 邻域的加权平均
Sobel X: [-1,0,1; -2,0,2; -1,0,1] 水平边缘
Sobel Y: [-1,-2,-1; 0,0,0; 1,2,1] 垂直边缘
🎯 卷积本质： 卷积核 = 一个"滑动窗口"，对每个像素及其邻居做加权求和。
不同的核 = 不同的特征提取器（边缘/模糊/锐化…）

2.2 Conway's Game of Life（康威生命游戏）
200ms
💡 点击画布上的格子可切换生死
🧬 生命游戏规则
每个细胞有 8 个邻居：
• 活细胞： 2-3 个活邻居 → 存活；否则 → 死亡
• 死细胞： 恰好 3 个活邻居 → 变为活细胞
// GLSL 生命游戏核心
int neighbors = sum3x3(grid);
float alive = grid[0][0];
float state = 0.0;
if (alive > 0.5) {
  state = (neighbors>=2 && neighbors<=3) ? 1.0 : 0.0;
} else {
  state = (neighbors == 3) ? 1.0 : 0.0;
}
🔬 生命游戏是图灵完备的： 可以模拟任意计算机！滑翔机 = "数据包"，碰撞 = "运算"

🌊 波动方程
离散形式（2D）：
buf2[y][x] = (buf[y-1][x] + buf[y+1][x] +
           buf[y][x-1] + buf[y][x+1]) * 0.5
         - buf_prev[y][x] * damping

// 简化版波动更新
float wave = (top + bottom + left + right) * 0.5

阻尼让波逐渐衰减到零。

2.5 Reaction-Diffusion（反应扩散系统）
0.040
0.060
1.0

🧪 Gray-Scott 反应扩散方程
两种化学物质 A 和 B：
∂A/∂t = D_A·∇²A - A·B² + f·(1-A)
∂B/∂t = D_B·∇²B + A·B² - (f+k)·B
f = feed rate（供应率）
k = kill rate（消耗率）
D_A, D_B = 扩散速率
// 离散更新
float reaction = A * B * B;
float newA = A + (DA * laplaceA - reaction
            + feed * (1.0 - A)) * dt;
float newB = B + (DB * laplaceB + reaction
            - (feed + kill) * B) * dt;
🔬 参数对图样的影响： 不同的 (f,k) 组合产生不同的斑图：斑点、条纹、迷宫、螺旋…
这就是 Turing 图灵斑图的起源！

💡 Phong 光照模型
I = I_ambient + I_diffuse + I_specular
环境光： k_a · I_a  (全局背景光)
漫反射： k_d · I_d · (N·L)  (Lambert 定律)
高光： k_s · I_s · (R·V)^α  (Blinn-Phong)
// GLSL Phong 着色
vec3 N = normalize(normal);
vec3 L = normalize(lightPos);
vec3 V = normalize(viewDir);
vec3 R = reflect(-L, N);
float diff = max(dot(N, L), 0.0);
float spec = pow(max(dot(R, V), 0.0), shininess);
🎯 光照三要素： 环境光 → 避免全黑
漫反射 → 表现物体立体感（Lambert 定律）
高光 → 表现材质光泽度

🗺️ 法线贴图原理
法线贴图 = 将法线编码为 RGB 颜色：
R = (Nx + 1) / 2  → 红色分量 = X 方向

3.3 Bump-maps（凹凸贴图）
15
45°
⛰️ 凹凸贴图 vs 法线贴图
Bump Map（高度图）：灰度图，亮度 = 高度
法线贴图：直接编码法线方向（RGB）

高度图 → 法线的计算：
dhdx = height(x+dx) - height(x-dx)
dhdy = height(y+dy) - height(y-dy)
normal = normalize(vec3(-dhdx, -dhdy, 1.0))
// 高度图求梯度得法线
float hL = texture(height, uv - dx).r;
float hR = texture(height, uv + dx).r;
float hD = texture(height, uv - dy).r;
float hU = texture(height, uv + dy).r;
vec3 normal = normalize(
  vec3(hL - hR, hD - hU, 0.05));
🎯 区别： 凹凸贴图 = 高度图（灰度），法线需计算
法线贴图 = 直接存储法线（彩色），省去计算
3.4 Ray Marching（光线步进）
20
💡 红色点 = 每一步的采样位置
🎯 Ray Marching 核心原理
SDF（有符号距离函数）：
每个点到场景中最近表面的距离

步进过程：
for (int i = 0; i < maxSteps; i++) {
  float d = sdf(rayPos);
  if (d < epsilon) → 命中！
  rayPos += rayDir * d;  // 安全步进 d
}
// 2D 圆形 SDF
float circleSDF(vec2 p, float r) {
  return length(p) - r;
}
🎯 每次步进距离 = SDF 值 → 保证不会"穿过"物体

🔦 Snell 定律 & Fresnel 效应
Snell 定律：
n₁·sin(θ₁) = n₂·sin(θ₂)

反射方向 (GLSL)：
vec3 R = reflect(I, N);

折射方向 (GLSL)：
vec3 T = refract(I, N, eta);
  // eta = n1/n2
// Fresnel 近似 (Schlick)
float fresnel = pow(1.0 - cosTheta, 5.0);
fresnel = mix(fresnel, 1.0, f0);
vec3 color = mix(refractColor,
         reflectColor, fresnel);
🎯 光滑表面同时发生反射和折射，
Fresnel 效应 = 掠射角时反射增强（如水面）
