🏎️ 弯道 ARC_CENTER 旋转原理 — 为什么模型中的"左下角"会变换位置？

🔑 关键问题解答

❓ 模型圆心在左下角，为什么旋转后到了右上角？

答：圆弧中心 ARC_CENTER 是在单元格局部坐标系中定义的。
它的值 (-CELL_HALF, +CELL_HALF) 在局部看是「左上角」（假设+X右，+Z下）。

当单元格旋转 90° 时，这个局部坐标被 2D 旋转矩阵旋转到世界坐标：

x' = ARC_CENTER_X·cos(θ) + ARC_CENTER_Z·sin(θ)
z' = -ARC_CENTER_X·sin(θ) + ARC_CENTER_Z·cos(θ)

对于 90° (θ=π/2, cos=0, sin=1)：
x' = 0·(-4.995) + 1·(4.995) = +4.995
z' = -1·(-4.995) + 0·(4.995) = +4.995

所以旋转后偏移变成了 (+CELL_HALF, +CELL_HALF)。
本质上：ARC_CENTER 在单元格的物理角点位置不变，但旋转的是单元格本身。
